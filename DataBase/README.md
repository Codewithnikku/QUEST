 # DataBase Folder
 
 SQL solutions for common relational problems. Run in MySQL-compatible engines.
 
 ## Index
 
 - ClassesWithAtLeast5Students.sql — count-based filtering on enrollments
 - CombineTwoTables.sql — basic LEFT JOIN across person and address
 - ConsecutiveNumbers.sql — identify streaks using self-join/window logic
 - CustomerWhoBoughtAllProducts.sql — division-style query with group counts
 - DepartmentHighestSalary.sql — department-wise top earners
 - EmployeeEarningMoreThanManagers.sql — employees earning more than their managers
 - FindCustomerReferee.sql — filter by referee presence
 - HumanTrafficOfStadium.sql — consecutive high-traffic days
 - InvestmentsIn2016.sql — yearly grouping and sums
 - MonthlyTransactionsI.sql — basic monthly aggregations
 - MovieRating.sql — user/movie average ratings
 - NotBoringMovies.sql — filter by odd-day and non-boring flag
 - ProductSalesAnalysisIII.sql — top sellers per product
 - RankScores.sql — ranking with dense rank
 - StudentsAndExamination.sql — attendance matrix via joins
 - TreeNode.sql — classify nodes as leaf/root/internal
 - TripsAndUsers.sql — trip cancellations and rates
 - UserActivityForlast30DaysI.sql — active users within 30 days
 - CUstomerPlacingLargestNumberofOrder.sql — customers with max orders
 
 ## Usage
 
 - Execute statements in your SQL client
 - Adapt table names to match your schema when necessary
