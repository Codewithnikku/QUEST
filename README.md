 # QUEST
 
 Curated solutions to Data Structures & Algorithms problems across multiple domains and languages.
 
 ## Structure
 
 - DSA — C++ implementations for classic algorithmic problems
 - DataBase — SQL queries for common relational problems
 - Math — JavaScript implementations for number theory and geometry problems
 
 ## Getting Started
 
 - Clone: `git clone https://github.com/Codewithnikku/QUEST.git`
 - Navigate into a folder and run solutions using the appropriate language toolchain
   - C++: `g++ file.cpp -o app && ./app`
   - JavaScript (Node.js): `node file.js`
   - SQL: execute in your SQL engine (tested on MySQL-compatible syntax)
